# CBDAS — Employee Attendance Dashboard

A React dashboard for managing and visualizing employee attendance data.

## Tech Stack

- React, Tailwind CSS, Recharts, Axios

## Features

- KPI cards (Total, Present, Absent, Late)
- Attendance charts (Line, Bar, Pie)
- Employee table with search, filter, sort, and pagination
- Dark mode, loading states, and error handling
- Reports page with export options

## Getting Started

```bash
npm install
npm start
```

The app runs on [http://localhost:3000](http://localhost:3000).

## API Integration

Set `REACT_APP_API_URL` to connect to a real backend. Falls back to mock data if not set.

## Production Build

```bash
npm run build
```
